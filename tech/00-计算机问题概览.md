## 计算机问题汇总

### 创建局域网

#### 创建步骤

- 以管理员身份，运行 【命令提示符】；
- 输入： `netsh wlan set hostednetwork mode=allow ssid=局域网的名字 key=密码`；
- 再输入： `netsh wlan start hostednetwork`；
- 当提示出现【已启动承载网络】，表示已成功；
- 打开网络连接，查看已经创建好的局域网。

#### 报错

##### 无法启动承载网络

输入 `netsh wlan start hostednetwork` 报错：

```shell
C:\WINDOWS\system32>netsh wlan start hostednetwork
无法启动承载网络。
组或资源的状态不是执行请求操作的正确状态。
```

原因：该无线驱动程序不支持承载网络。

**解决：** 更新网卡的驱动程序。

检查网卡的驱动程序：`netsh wlan show drivers`，查看到 "支持的承载网络  : 否"。

```shell
C:\WINDOWS\system32>netsh wlan show drivers
接口名称: WLAN

    驱动程序                  : Realtek 8822CE Wireless LAN 802.11ac PCI-E NIC
    供应商                    : Realtek Semiconductor Corp.
    提供程序                  : Realtek Semiconductor Corp.
    日期                      : 2020/8/13
    版本                      : 2024.0.8.127
    INF 文件                  : oem125.inf
    类型                      : 本机 WLAN 驱动程序
    支持的无线电类型          : 802.11n 802.11g 802.11b 802.11ac 802.11n 802.11a
    支持 FIPS 140-2 模式: 是
    支持 802.11w 管理帧保护 : 是
    支持的承载网络  : 否
    ...
```

在计算机 -> 属性 -> 设备管理器 -> 网络适配器，在 “Realtek 8822CE Wireless LAN 802.11ac PCI-E NIC” 上右键再点 "更新驱动程序 -> 浏览我的电脑以查找驱动程序"，选择比较旧的版本进行更新（从网上下载）。
